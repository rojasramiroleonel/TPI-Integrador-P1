# TPI Integrador P1

El programa permite:

- Agregar países.
- Actualizar información de países.
- Buscar países.
- Filtrar países.
- Ordenar países.
- Mostrar estadísticas.

Archivos del proyecto:

- main.py - Programa principal
- archivo.py - Manejo de archivos CSV
- paises.py - Gestión de países
- estadisticas.py - Cálculo de estadisticas
- paises.csv - Base de datos de países

Para ejecutar el programa: main.py
